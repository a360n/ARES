# ARES: Autonomous Rescue & Emergency System

ARES is a hybrid robotic system (UAV + UGV) designed for disaster response, environmental monitoring, and autonomous exploration.

## Getting Started
### Prerequisites
- Python 3.11 or higher
- Git
- Virtual Environment (recommended)

### Quick Installation (Windows PowerShell)
1. Clone the repository:
   `git clone https://github.com/a360n/ARES.git`
   `cd ARES`

2. Create and activate a virtual environment:
   `python -m venv venv`
   `.\venv\Scripts\Activate.ps1`

3. Install dependencies:
   `pip install -r requirements.txt`

4. Run the server:
   `python app.py`
